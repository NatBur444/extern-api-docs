API позволяет интегрировать сервис отчетности с контролирующими органами в ваши процессы и информационные системы.

Экстерн является сервисом отчетности, который обеспечивает `электронный документооборот`_ между хозяйствующими субъектами и контролирующими органами. Веб сервис обладает рядом преимуществ, но не всегда может поддержать все сценарии пользователя. API Контур Экстерна нужен, чтобы встроить работу с электронным документооборотом отчетности в интерфейсы информационных систем клиента.

Возможности
===========

1. Отправка отчетности в контролирующие органы, такие как ФНС, ФСС, ПФР, Росстат.
2. Централизованный мониторинг отчетности в сложных и распределенных структурах внутри своей информационной системы.
3. Автоматизация работы с требованиями – массовая загрузка поступивших требований в информационную систему клиента, отправка документов в ответ на требование без ограничения на количество и объем отправляемых документов.
4. Организация зашифрованного или расшифрованного архива отчетов в собственной информационной системе или локальном хранилище.

Модели сотрудничества
=====================

Абонент Контур.Экстерна
-----------------------

Организация, которая хочет автоматизировать процессы отчетности внутри компании и предоставить возможность бухгалтерам работать в привычных интерфейсах корпоративной информационной системы.

Использование API Экстерна поможет решить следующие задачи:

   * контроль процессов по работе с отчетностью для организаций с большим количеством ЮЛ в организационной структуре;
   * оптимизация работы с отчетностью большого количества пользователей;
   * полная или частичная автоматизация работы с потоком новых требований ФНС;
   * необходимость отправлять большое количество документов из корпоративной информационной системы в ответ на требования ФНС;
   * организация локального архива (хранилища) отправленной отчетности, входящих писем и требований ФНС.

Компания-партнер Контура
------------------------

Компания, которая разрабатывает или предоставляет своим клиентам различные информационные системы и хочет внедрить в них электронный документооборот с контролирующими органами. Таким образом, вендору, который хочет расширить функциональность своего продукта в части электронного документооборота с контролирующими органами, не нужно самостоятельно получать статус оператора ЭДО.

.. note:: Вам необходимо сообщать пользователям, что они будут использовать Контур.Экстрен.
