Для работы с API Контур.Экстерна необходимо в каждом запросе указывать :doc:`/manuals/auth.sid` для аутентификации пользователя, который работает с API. В этом разделе мы расскажем о том, какие существуют способы получения auth.sid. Отдельно стоит отметить, что аутентификация для всех продуктов Контура единая, поэтому получение auth.sid происходит вне API Контур.Экстерна, а в отдельном API Аутентификатора:

- **Тестовая площадка**: https://api.testkontur.ru/auth/v5.11 

- **Рабочая площадка**: https://api.kontur.ru/auth/v5.11 

Существует два возможных способа аутентификации: по сертификату и доверительная.
