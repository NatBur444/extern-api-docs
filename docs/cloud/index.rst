=====================

Данный раздел посвящен возможностям API по работе с облачной криптографией. Облачная криптография удобна тем, что криптографические операции, требующие работы с закрытым ключом (подписание, расшифровка), выполняются не на рабочем месте пользователя, а на сервере Удостоверяющего центра, выдавшего сертификат облачной электронной подписи. Закрытый ключ хранится на сервере Удостоверяющего центра, доступ к нему пользователь дает посредством ввода пароля, полученного в смс-сообщении на телефон, указанный в заявлении на выдачу сертификата. Таким образом существенно снижаются требования к программному обеспечению на рабочем месте пользователя: нет необходимости установки и настройки криптопровайдеров и закрытых ключей сертификатов электронной подписи, необходимо только в момент подписания или расшифровки документа ввести код из полученного смс-сообщения.
